## What is CanAccesible?

**CanAccesible** is a comprehensive web platform designed for the Canary Islands that empowers citizens to actively participate in improving accessibility across the archipelago. Built with a full React + Express stack, it combines modern web technology with social awareness to create a collaborative accessibility community.

## Platform architecture

CanAccesible is a full-stack monorepo with a React 19 frontend and an Express 5 backend:

```
CanAccesible/
├── frontend/    # React 19 + Vite + TailwindCSS 4
├── backend/     # Express 5 + Sequelize + MySQL
├── docs/        # Project documentation
└── ldap/        # OpenLDAP configuration
```

The backend exposes a REST API consumed by the React frontend and an admin dashboard rendered server-side using EJS templates.

## Live application

The platform is publicly accessible at [canaccesible.es](https://canaccesible.es). The interactive API documentation is available at [canaccesible.es/api-docs](https://canaccesible.es/api-docs).
